# TaskFlow Pro

TaskFlow Pro is a lightweight task management web app built with React and Vite.

## Run locally

Prerequisites: Node.js (LTS recommended)

1. Install dependencies:

   npm install

2. Start the dev server:

   npm run dev

3. Build for production:

   npm run build

This project does not require any external AI service keys. Any previous references to AI platforms or Gemini have been removed.
